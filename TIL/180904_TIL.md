# 컴퓨터의 역사

### 전기로 동작하는 스위치 : 릴레이
### **스위치에 의해서 작동된다**
    - 내부가 수많은 스위치로 이루어져 있기 때문에
    - 0과 1을 저장하여 동작
    - 내부 동작원리가 같다

스위치가 3개면 2의 3승 개의 수를 표현 할 수 있는 것
> 릴레이 1개 = 1비트 = 두 가지 경우의 수 (0과 1)<br>
**1바이트 = 8비트** = 256 가지 경우의 수(2의 8승)


    바이트가 1000개면 > 1킬로바이트

    바이트가 1024개면 > 1키비바이트
